⚠️ Disclaimer

MedWatch is an educational tool that retrieves data from the FDA Adverse Event Reporting System (FAERS). Adverse event reports do not prove that a drug caused the reported reaction. The data should not be used to make medical decisions. Always consult a healthcare professional for medical advice.

🛠️ Tech Stack

Built with Python, Flask, HTML/CSS, Bootstrap, JavaScript, Plotly for data visualization, the openFDA REST API for adverse event data, and Google Gemini for AI-powered insights.
